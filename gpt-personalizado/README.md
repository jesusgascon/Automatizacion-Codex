# GPT personalizado

Carpeta preparada para crear un GPT orientado a documentar automatizaciones locales.

## Archivos

- `instructions.txt`
  Instrucciones principales del GPT.
- `welcome-message.txt`
  Mensaje de bienvenida sugerido.
- `conversation-starters.txt`
  Iniciadores de conversacion listos para pegar.
- `knowledge-files.txt`
  Lista de documentos recomendados para subir como conocimiento.

## Uso recomendado

1. Crear un GPT nuevo.
2. Usar como nombre:
   - `Documentador de Automatizaciones Locales`
3. Pegar `instructions.txt` en las instrucciones.
4. Pegar `welcome-message.txt` como mensaje inicial.
5. Copiar los iniciadores de `conversation-starters.txt`.
6. Subir los documentos listados en `knowledge-files.txt`.

La version extensa y razonada sigue estando en:

```text
docs/gpt-personalizado-documentador.md
```

La lista de conocimiento incluye tambien `docs/configuracion-y-mantenimiento.md` para que el GPT pueda documentar no solo instalacion y arquitectura, sino tambien operacion recurrente y actualizaciones.
